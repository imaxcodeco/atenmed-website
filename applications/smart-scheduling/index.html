🔗 Integração Google Calendar
⚠️ Atenção: É necessário autenticar com o Google Calendar para utilizar o sistema de agendamento.

Como configurar:

  1. Configure as credenciais OAuth2 no Google Cloud Console
  2. Adicione as variáveis de ambiente no arquivo .env
  3. Clique no botão abaixo para autenticar
  4. Autorize o acesso ao Google Calendar
  5. Crie calendários compartilhados para cada médico
