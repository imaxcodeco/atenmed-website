=== COMANDOS PARA EXECUTAR NO SERVIDOR ===

1. Conectar ao servidor:
ssh -i C:\Users\Ian_1\Documents\AtenMed\site-atenmed.pem ubuntu@3.129.206.231

2. Ir para diretório do projeto:
cd /var/www/atenmed

3. Testar emails:
node scripts/test-email-ses.js

4. Verificar logs PM2:
pm2 logs atenmed --lines 50

5. Verificar status:
pm2 status
pm2 info atenmed

6. Restart se necessário:
pm2 restart atenmed
